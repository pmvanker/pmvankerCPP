# C++20 Features

C++20, released in 2020, introduced major features like modules, concepts, and coroutines. This project currently has no examples, but placeholders for future additions.

## Key Features Added in C++20

- **Modules**: Replace header files with modules.
- **Concepts**: Constraints on templates.
- **Coroutines**: For asynchronous programming.
- **Ranges**: Enhanced algorithms.
- **Three-Way Comparison**: `<=>` operator.
- **Calendar and Time Zone**: Improved chrono.
- **Atomic Smart Pointers**: Thread-safe smart pointers.

This project includes examples for ranges, concepts, format, span, and jthread.
