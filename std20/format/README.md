# std::format

## Explanation
Type-safe string formatting.

## Use Case
- Replace printf with safety.

## How to Use
```cpp
std::format("Hello, {}", name);
```

## Methods
- Format strings with {} placeholders.

## Benefits
- Type-safe, no buffer overflows.

## Cons
- Requires C++20 support.

See `format_example.cpp` for basic formatting.
