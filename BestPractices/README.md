# Best Practices in Modern C++

This folder contains examples of good practices, common mistakes to avoid, and key things to take care of in C++ programming.

## Topics
- **RAII**: Resource Acquisition Is Initialization.
- **Exception Safety**: Handling errors safely.
- **Const Correctness**: Proper use of const.

Each subfolder has examples of good and bad code, explanations of what to avoid, and tips.
