# C++17 Features

C++17, released in 2017, focused on code simplicity and performance improvements.

## Key Features Added in C++17

- **Filesystem**: Library for file and directory operations.
- **Optional**: For values that may or may not be present.
- **Variant**: Type-safe union.
- **Any**: Type-safe container for any type.
- **String View**: Non-owning string reference.
- **Structured Bindings**: Unpack tuples/pairs.
- **Fold Expressions**: For variadic templates.
- **constexpr if**: Compile-time conditionals.

This project includes examples for filesystem, optional, variant, any, and string_view.
