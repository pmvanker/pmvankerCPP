# C++14 Features

C++14, released in 2014, built on C++11 with bug fixes, performance improvements, and new features for better usability.

## Key Features Added in C++14

- **Enhanced Constexpr**: Relaxed restrictions for compile-time functions.
- **Generic Lambdas**: Lambdas with auto parameters.
- **Variable Templates**: Templates for variables.
- **Aggregate Member Initialization**: Improved initialization.
- **Chrono Literals**: User-defined literals for time (e.g., `5s`, `300ms`).
- **Binary Literals**: `0b` prefix for binary numbers.
- **Digit Separators**: `'` for readability in numbers.
- **std::make_unique**: For creating unique_ptr.
- **Deprecation of Some Features**: Like std::auto_ptr.

This project includes examples for chrono literals, std::make_unique, and generic lambdas in their respective directories.
