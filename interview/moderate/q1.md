# Question 1: Explain STL containers: vector vs map.

## Problem
Describe vector and map, when to use each.

## Solution
- **Vector**: Dynamic array, contiguous memory, fast access O(1), insert/delete O(n).
- **Map**: Ordered associative, key-value, O(log n) access.

### When to Use
- Vector: Sequential access, resizing.
- Map: Sorted key lookup.

### Code Example
```cpp
#include <vector>
#include <map>
int main() {
    std::vector<int> v = {1, 2, 3};
    std::map<int, std::string> m = {{1, "one"}};
    return 0;
}
```
