# Question 50: What is the spaceship operator in C++20?

**What is the spaceship operator in C++20?**
    - **Answer/Explanation:** The spaceship operator `<=>` performs three-way comparison.
    - **Code Example:**
      ```cpp
      auto cmp = (a <=> b);
      if (cmp < 0) {}  // a < b
      ```
    - **Key Concepts:** Comparison operators, consistency.
    - **Problem-Solving Technique:** Simplifying comparison logic.
