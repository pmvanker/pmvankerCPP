# Question 12: What are lambda expressions?

**What are lambda expressions?**
    - **Answer/Explanation:** Lambdas are anonymous functions that can capture variables from their scope.
    - **Code Example:**
      ```cpp
      auto lambda = [](int x) { return x * 2; };
      std::cout << lambda(5);
      ```
    - **Key Concepts:** Anonymous functions, closures.
    - **Problem-Solving Technique:** Functional programming.
