# Question 22: What is a race condition?

**What is a race condition?**
    - **Answer/Explanation:** A race condition occurs when multiple threads access shared data concurrently, leading to unpredictable behavior.
    - **Code Example:** Multiple threads incrementing a shared counter without synchronization.
    - **Key Concepts:** Concurrency issues.
    - **Problem-Solving Technique:** Synchronization primitives.
