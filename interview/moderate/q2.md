# Question 2: Implement a stack using two queues.

## Problem
Implement a stack data structure using two queues. Provide push, pop, top, and empty operations.

## Solution
Use two queues: q1 for main, q2 for temp. For push, add to q1. For pop, move all but last from q1 to q2, pop last, swap queues.

### Code Example
```cpp
#include <queue>

class MyStack {
    std::queue<int> q1, q2;
public:
    void push(int x) {
        q1.push(x);
    }
    
    int pop() {
        while (q1.size() > 1) {
            q2.push(q1.front());
            q1.pop();
        }
        int top = q1.front();
        q1.pop();
        std::swap(q1, q2);
        return top;
    }
    
    int top() {
        while (q1.size() > 1) {
            q2.push(q1.front());
            q1.pop();
        }
        int top = q1.front();
        q2.push(top);
        q1.pop();
        std::swap(q1, q2);
        return top;
    }
    
    bool empty() {
        return q1.empty();
    }
};
```

### Key Concepts
- Queue operations, stack simulation.

### Problem-Solving Technique
- Think about how to reverse order using queues.
