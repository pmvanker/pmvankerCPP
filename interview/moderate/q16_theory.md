# Question 16: What are initializer lists in C++11?

**What are initializer lists in C++11?**
    - **Answer/Explanation:** Initializer lists allow uniform initialization syntax for objects.
    - **Code Example:**
      ```cpp
      std::vector<int> v = {1, 2, 3};
      int arr[] = {1, 2, 3};
      ```
    - **Key Concepts:** Uniform initialization.
    - **Problem-Solving Technique:** Consistent object creation.
