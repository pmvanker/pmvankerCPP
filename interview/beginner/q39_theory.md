# Question 39: Explain maps in C++.

**Explain maps in C++.**
    - **Answer/Explanation:** Maps are associative containers that store key-value pairs, sorted by key.
    - **Code Example:**
      ```cpp
      #include <map>
      std::map<std::string, int> m;
      m["key"] = 10;
      ```
    - **Key Concepts:** Key-value storage, ordering.
    - **Problem-Solving Technique:** Associative data access.
