# Question 31: Explain the `new` and `delete` operators.

**Explain the `new` and `delete` operators.**
    - **Answer/Explanation:** `new` allocates memory on the heap, `delete` deallocates it.
    - **Code Example:**
      ```cpp
      int* ptr = new int(10);
      delete ptr;
      ```
    - **Key Concepts:** Dynamic memory, heap allocation.
    - **Problem-Solving Technique:** Manual memory management.
