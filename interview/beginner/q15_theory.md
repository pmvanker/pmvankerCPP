# Question 15: Explain the `auto` keyword in C++11.

**Explain the `auto` keyword in C++11.**
    - **Answer/Explanation:** `auto` allows automatic type deduction for variables based on their initializer.
    - **Code Example:**
      ```cpp
      auto x = 5;  // int
      auto y = 3.14;  // double
      ```
    - **Key Concepts:** Type inference.
    - **Problem-Solving Technique:** Simplifying variable declarations.
