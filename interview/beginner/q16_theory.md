# Question 16: What are comments in C++?

**What are comments in C++?**
    - **Answer/Explanation:** Comments are non-executable text for documentation. Single-line (//) and multi-line (/* */).
    - **Code Example:**
      ```cpp
      // This is a single-line comment
      /* This is a
         multi-line comment */
      ```
    - **Key Concepts:** Code documentation.
    - **Problem-Solving Technique:** Writing readable code.
