# Question 3: Reverse a linked list.

## Problem
Given the head of a singly linked list, reverse it and return the new head.

## Solution
Iterative: Use three pointers - prev, curr, next. Reverse links.

### Code Example
```cpp
struct ListNode {
    int val;
    ListNode* next;
    ListNode(int x) : val(x), next(nullptr) {}
};

ListNode* reverseList(ListNode* head) {
    ListNode* prev = nullptr;
    ListNode* curr = head;
    while (curr) {
        ListNode* next = curr->next;
        curr->next = prev;
        prev = curr;
        curr = next;
    }
    return prev;
}
```

### Key Concepts
- Linked list manipulation, pointers.

### Problem-Solving Technique
- Draw the list, simulate the reversal step by step.
