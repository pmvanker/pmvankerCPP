# Question 11: Explain arrays in C++.

**Explain arrays in C++.**
    - **Answer/Explanation:** Arrays are fixed-size collections of elements of the same type stored in contiguous memory locations.
    - **Code Example:**
      ```cpp
      int arr[5] = {1, 2, 3, 4, 5};
      std::cout << arr[0];  // access first element
      ```
    - **Key Concepts:** Fixed size, indexing.
    - **Problem-Solving Technique:** Array manipulation.
