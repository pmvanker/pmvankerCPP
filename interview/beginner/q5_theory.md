# Question 5: What are variables and how are they declared in C++?

**What are variables and how are they declared in C++?**
   - **Answer/Explanation:** Variables are named storage locations in memory. They are declared by specifying the data type followed by the variable name.
   - **Code Example:**
     ```cpp
     int age = 25;
     std::string name = "John";
     ```
   - **Key Concepts:** Variable declaration, scope.
   - **Problem-Solving Technique:** Proper naming and initialization.
