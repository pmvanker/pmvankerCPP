# Question 12: What are strings in C++?

**What are strings in C++?**
    - **Answer/Explanation:** Strings are sequences of characters. C++ has C-style strings (char arrays) and std::string from the STL.
    - **Code Example:**
      ```cpp
      #include <string>
      std::string str = "Hello";
      char cstr[] = "World";
      ```
    - **Key Concepts:** String handling, null termination.
    - **Problem-Solving Technique:** String operations.
