# Question 1: What is C++?

**What is C++?**
   - **Answer/Explanation:** C++ is a general-purpose, object-oriented programming language developed by Bjarne Stroustrup as an extension of the C programming language. It supports multiple programming paradigms including procedural, object-oriented, and generic programming.
   - **Code Example:**
     ```cpp
     #include <iostream>
     int main() {
         std::cout << "Hello, World!" << std::endl;
         return 0;
     }
     ```
   - **Key Concepts:** Programming paradigms, language evolution.
   - **Problem-Solving Technique:** Understanding language fundamentals.
