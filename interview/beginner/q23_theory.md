# Question 23: Explain the concept of inheritance in C++.

**Explain the concept of inheritance in C++.**
    - **Answer/Explanation:** Inheritance allows a class to inherit properties and methods from another class.
    - **Code Example:**
      ```cpp
      class Animal {};
      class Dog : public Animal {};
      ```
    - **Key Concepts:** Base and derived classes, polymorphism.
    - **Problem-Solving Technique:** Code reuse.
