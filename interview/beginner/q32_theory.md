# Question 32: What are pointers in C++?

**What are pointers in C++?**
    - **Answer/Explanation:** Pointers store memory addresses of variables.
    - **Code Example:**
      ```cpp
      int x = 10;
      int* ptr = &x;
      std::cout << *ptr;  // dereference
      ```
    - **Key Concepts:** Memory addresses, indirection.
    - **Problem-Solving Technique:** Direct memory access.
