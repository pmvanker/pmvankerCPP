# Question 17: Explain the `sizeof` operator.

**Explain the `sizeof` operator.**
    - **Answer/Explanation:** `sizeof` returns the size in bytes of a type or variable.
    - **Code Example:**
      ```cpp
      std::cout << sizeof(int);  // typically 4
      ```
    - **Key Concepts:** Memory size, platform dependency.
    - **Problem-Solving Technique:** Memory calculations.
