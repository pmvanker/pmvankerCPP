# Question 42: What are templates in C++?

**What are templates in C++?**
    - **Answer/Explanation:** Templates enable generic programming by allowing functions and classes to work with different types.
    - **Code Example:**
      ```cpp
      template <typename T>
      T add(T a, T b) { return a + b; }
      ```
    - **Key Concepts:** Generic programming, type parameters.
    - **Problem-Solving Technique:** Code reusability.
