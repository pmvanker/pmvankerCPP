# Question 18: What is type casting in C++?

**What is type casting in C++?**
    - **Answer/Explanation:** Type casting converts one data type to another. C++ has implicit and explicit casting.
    - **Code Example:**
      ```cpp
      int x = 10;
      double y = (double)x;  // explicit cast
      ```
    - **Key Concepts:** Type conversion, safety.
    - **Problem-Solving Technique:** Avoiding data loss.
