# Question 8: Explain control structures in C++ (if-else, loops).

**Explain control structures in C++ (if-else, loops).**
   - **Answer/Explanation:** Control structures direct the flow of execution. `if-else` for conditional execution, `for`, `while`, `do-while` for loops.
   - **Code Example:**
     ```cpp
     if (x > 0) {
         std::cout << "Positive";
     } else {
         std::cout << "Non-positive";
     }
     for (int i = 0; i < 5; i++) {
         std::cout << i << " ";
     }
     ```
   - **Key Concepts:** Conditional statements, iteration.
   - **Problem-Solving Technique:** Flow control logic.
