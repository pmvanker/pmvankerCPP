# C++ Interview Questions

This folder contains top 100+ commonly asked C++ interview questions, categorized by difficulty: Beginner, Moderate, Hard.

The questions are divided into:
- **Theory**: Conceptual questions with explanations
- **Coding**: Programming problems with solutions

Each question includes:
- Problem statement
- Solution explanation
- Code example
- Key concepts and problem-solving techniques

## Categories
- **Beginner**: Basics like syntax, OOP, pointers.
- **Moderate**: STL, memory management, concurrency.
- **Hard**: Advanced topics like templates, optimization, design patterns.

## File Structure
- Theory questions: `q[number]_theory.md`
- Coding problems: `q[number]_coding.md`

## Problem-Solving Techniques
- **Understand the Problem**: Read carefully, identify inputs/outputs.
- **Plan**: Think of edge cases, time/space complexity.
- **Code**: Write clean, efficient code.
- **Test**: Check with examples, edge cases.
- **Optimize**: Improve if needed.

Use these for preparation. Questions are based on common sources like LeetCode, GeeksforGeeks, etc.
