# Question 1: Explain template metaprogramming.

## Problem
What is TMP? Example.

## Solution
Compile-time programming with templates.

### Example: Factorial
```cpp
template<int N>
struct Factorial {
    static const int value = N * Factorial<N-1>::value;
};
template<>
struct Factorial<0> { static const int value = 1; };

int main() {
    std::cout << Factorial<5>::value << std::endl; // 120
}
```

### Key Concepts
- Recursive templates, constexpr.

### Problem-Solving Technique
- Think recursively for compile-time computation.
