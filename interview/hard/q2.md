# Question 2: Implement a thread-safe singleton.

## Problem
Implement a singleton class that is thread-safe using C++11 features.

## Solution
Use std::call_once with std::once_flag for lazy initialization.

### Code Example
```cpp
#include <mutex>
#include <memory>

class Singleton {
private:
    static std::unique_ptr<Singleton> instance;
    static std::once_flag flag;
    
    Singleton() {}
    
public:
    static Singleton* getInstance() {
        std::call_once(flag, []() {
            instance.reset(new Singleton());
        });
        return instance.get();
    }
    
    // Other methods
};

std::unique_ptr<Singleton> Singleton::instance;
std::once_flag Singleton::flag;
```

### Key Concepts
- Thread safety, lazy initialization, smart pointers.

### Problem-Solving Technique
- Avoid double-checked locking pitfalls, use standard library features.
